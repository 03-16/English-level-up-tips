# Loved 爱过

我牵过许多女孩的手

也搂过许多女孩的腰

我看过许多云彩

躺在不同的女孩腿上睡着

我去过许多城市

在不同的女孩身边醒来

但我知道

我是个 `Good Boy`


你要问我 

我究竟是害怕孤独

还是渴望爱情

我不敢确定


端一杯茶 

拾一本书


我想对你说 

我，也曾**爱过**
